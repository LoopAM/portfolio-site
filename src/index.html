Projects

Here are a few of the things I've been working on recently

Mood Track

Mood Track uses the Spotify API to read the user's most recent five songs and then recommends a song similar to the mood of those last five songs. The app also gives an option to search for the songs if the user doesn't have a Spotify accout.

The app uses NodeJS with ExpressJS to handle the requests and HandlebarsJS to handle the HTML rendering on the server.

Pokemon Memory Game

Pokemon Memory Game is a simple memory game inspired by the Pokemon franchise.

The app was built using ReactJS and Webpack.
